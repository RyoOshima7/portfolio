# Career（就活）

## 目的
就活で使う職務要約、面接ストーリー、企業研究を整理し、自分の経験と志望理由を他人に伝わる形で残すことを目的としています。

## 使用技術
Markdown, STAR法, キャリア整理, 企業研究

## 実行方法
1. `resume/` で職務要約やスキル棚卸しを整理します。
2. `interview_stories/` でSTAR形式のエピソードを蓄積します。
3. `company_notes/` で企業ごとの課題仮説や志望動機の材料をまとめます。

## 学び
- 就活資料は、自分が理解しているだけでなく、相手が短時間で理解できる形にする必要があります。
