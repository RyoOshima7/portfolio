# BI / Visualization（可視化・ダッシュボード）

## 目的
意思決定に使える見せ方を学び、ダッシュボード設計や計測設計を成果物として整理することを目的としています。

## 使用技術
スプレッドシート, BI設計, KPI設計, ダッシュボード設計, Markdown

## 実行方法
1. `dashboards/` で画面設計、KPI、使い方、改善点を確認します。
2. `tracking_plan/` でイベント名、プロパティ、発火条件、目的を整理します。
3. 可視化成果物を追加する際は、見る人が指標の意味を理解できる説明を付けます。

## 学び
- 可視化は見た目だけでなく、何を判断するための画面かを明確にする必要があります。
