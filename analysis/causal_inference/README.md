# Causal Inference（因果推論）

## 目的
相関と因果の違いを説明できる状態を目指し、因果推論の基礎概念と簡単な演習を整理することを目的としています。

## 使用技術
因果推論基礎, DAG, バイアス, 交絡, Markdown, Python（必要に応じて）

## 実行方法
1. `concepts.md` で用語や考え方を整理します。
2. `exercises/` で小さな例題を手計算やPythonで確認します。
3. `onepager.md` に説明用の要約を作成し、相関と因果の違いを短く話せるようにします。

## 学び
- 分析結果をそのまま因果とみなさない姿勢が重要です。
