# Product Analytics（プロダクト分析）

BtoB SaaSプロダクトアナリスト向けに、**問い → 仮説 → 指標 → 検証 → 示唆** の型でミニPJを蓄積します。

## 置くもの（テンプレ）
- `README.md`：目的 / データ前提 / 使う指標 / 結果 / 示唆 / 次改善
- `queries/`：SQL（必要なら）
- `notes.md`：迷い・読み違い・学び（面接で語れる材料）

## 例（作ると強いテーマ）
- ファネル（Activation / Conversion）
- 継続（Retention / Cohort）
- 解約（Churnの兆候仮説）
