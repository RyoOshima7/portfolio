# Analysis

このディレクトリには、**分析ミニプロジェクト**（問い → 仮説 → 指標定義 → SQL/集計 → 解釈 → 示唆）を残します。

## 目的
- BtoB SaaS のプロダクト分析でよく使う型（ファネル、継続率、解約、アクティベーション等）を、再現可能な形でまとめる
- 「SQLが書ける」だけでなく、**意思決定につながる示唆**まで出せることを証明する

## 予定しているアウトプット例
- tracking_plan.md（イベント設計 / KPI定義）
- funnel_analysis.md（ファネル分析）
- retention_analysis.md（継続率/リテンション）
- churn_hypotheses.md（解約仮説と検証）
