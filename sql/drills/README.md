# SQL Drills

SQLの基礎〜実務で頻出のパターンを、**1問 = 1ファイル**で整理して積み上げています。  
目的は「学習したこと」ではなく、**就活で “何ができるか” が一目で伝わる形**にすることです。

---

## ルール
- ファイル名は `q###_topic.sql`（連番 + 内容）
- 各ファイルは以下の構成
  - 問題（要件）
  - 解答SQL
  - Learnings（学びを2〜3行で要約）
