# SQL Drills

このディレクトリは、SQL の基礎力を体系的に身につけることを目的とした演習ログです。  
単にクエリを書くだけでなく、**問題定義 → 実装 → 振り返り**の流れを一貫して記録しています。

---

## 目的

- SQL の基本構文を正確に使えるようになること
- 条件指定・文字列検索など、実務で頻出するパターンを整理すること
- 自分の思考過程（判断基準・迷った点）を言語化し、再現可能な形で残すこと

---

## ディレクトリ構成

```text
sql/drills/
├─ 2026-02/
│  ├─ q001_club_in_alias/
│  │  ├─ 01_problem.md
│  │  ├─ 02_answer.sql
│  │  └─ 03_notes.md
│  ├─ q002_club_name_like_ou/
│  └─ q003_address_like_not/
└─ README.md
````

各問題は、以下の構成で統一しています。

* **01_problem.md**
  問題定義・条件・出力要件（仕様）
* **02_answer.sql**
  SQL 実装（先頭に要約コメントを記載）
* **03_notes.md**
  学び・判断基準・詰まりポイントの整理

---

## 学習方針

* 問題文をそのまま SQL に落とすのではなく、
  **日本語の条件を分解してから WHERE 句を組み立てる**
* 可読性を重視し、
  別名（AS）、インデント、条件の並び順を意識する
* 「なぜそう書いたか」「どこで迷ったか」を必ず notes.md に残す

---

## これまでに扱った主な内容

* SELECT / WHERE の基本構文
* AS による別名指定
* IN / OR を使った複数条件指定
* LIKE / NOT LIKE による文字列検索
* AND / OR の組み合わせと可読性の考え方

---

## このログで意識していること

* 正解を書くことよりも、**再現できる考え方を残す**
* 数か月後に見返しても、**判断理由が分かる状態**にする
* 実務でのデータ抽出・分析業務を想定した書き方をする
