- `q017_exists_under_60.sql` : EXISTS / 相関サブクエリ（60点未満が1科目でもある学生の抽出）
- `q018_left_join_no_affiliation.sql` : LEFT JOIN / IS NULL（どの部活にも所属していない学生の抽出）
