# 2026-02 SQL Drills

2026年2月に実施した SQL 基礎演習の記録。  
SELECT 文と条件指定を中心に、SQL の基本的な考え方を整理した。

---

## 実施内容

### q001_club_in_alias
- AS を使った別名指定
- 同一カラムに対する複数条件の指定（IN）

### q002_club_name_like_ou
- LIKE を使った部分一致検索
- OR 条件による複数パターン指定

### q003_address_like_not
- LIKE / NOT LIKE を使った肯定・否定条件
- AND による複数条件の組み合わせ
