# Roadmap（就活ロードマップ）

このフォルダは、就活ロードマップの **原本（マスター/月次/ガント）** をリポジトリ内で参照できるようにしたものです。

- `master.md`：マスター版（ID / カテゴリ / Proof / Why / How）
- `monthly_plan.md`：月次計画（毎月のゴール・タスク・Done）
- `gantt_month.csv`：月次ガント（更新用データ）

## 使い方（おすすめ）
1. 取り組むIDを1つ決める（例：SQL 2026-02）
2. そのIDのProofを、このリポジトリ内の該当フォルダに **リンクで示せる形** にする
3. 週次で `README.md` の更新ログに「何を作ったか」を1行で残す
