# q001_sales_total - 売上CSVの総売上計算

## 目的
CSVを読み込み、`quantity × unit_price` で総売上を計算する基礎を身につけることを目的としています。

## 使用技術
Python, csv, for文, 関数, 例外処理, 数値変換

## 実行方法
1. `learning/main.py` を自力で完成させます。
2. `completed/main.py` と見比べて差分を確認します。
3. 必要なら `python learning/main.py` または `python completed/main.py` を実行し、結果を確認します。
4. `docs/problem.md` と `docs/requirements.md` を見返し、何を学ぶ課題か説明できるようにします。

## 学び
- CSVから値を読み取り、計算して合計を出す流れは、分析処理の基本になります。
  
## SaaS企業での活用

CSVの各行から金額を計算して合計する処理は、SaaS企業で請求データや契約データを集計し、月次売上を把握するための基礎となります。集計結果は、売上の推移や予算との差を確認し、追加調査が必要な期間を判断する際に役立ちます。
