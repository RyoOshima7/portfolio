# q007_inventory_total - 在庫一覧と合計金額表示

## 目的
商品一覧を表示し、在庫金額の合計を出す。

## 使う内容
リスト, for文, 関数, 計算

## 学習の進め方
1. learning/main.py を自力で完成させる
2. completed/main.py と見比べる
3. README に「工夫した点」を追記する

## 実行例
商品一覧
商品名: Apple, 単価: 120円, 在庫数: 10
商品名: Banana, 単価: 80円, 在庫数: 15
商品名: Orange, 単価: 100円, 在庫数: 8
合計在庫金額: 3200円
