# q007_inventory_total - 在庫一覧と合計金額表示

## 目的
商品一覧を表示し、在庫金額の合計を計算する基礎を身につけることを目的としています。

## 使用技術
Python, リスト, for文, 関数, 計算

## 実行方法
1. `learning/main.py` を自力で完成させます。
2. `completed/main.py` と見比べて差分を確認します。
3. 必要なら `python learning/main.py` または `python completed/main.py` を実行します。
4. リスト構造から値を取り出して計算する流れを確認します。

## 学び
- 一覧表示と合計計算を組み合わせることで、データ構造の扱いに慣れられます。
