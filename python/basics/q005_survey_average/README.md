# q005_survey_average - アンケート満足度平均

## 目的
満足度の平均を計算し、未入力値を除外して集計する基礎を身につけることを目的としています。

## 使用技術
Python, csv, if文, 関数, 例外処理

## 実行方法
1. `learning/main.py` を自力で完成させます。
2. `completed/main.py` と見比べて差分を確認します。
3. 必要なら `python learning/main.py` または `python completed/main.py` を実行します。
4. 未入力を数えない理由を docs と合わせて確認します。

## 学び
- 平均を計算するときは、欠損値をどう扱うかを先に決めることが重要です。
