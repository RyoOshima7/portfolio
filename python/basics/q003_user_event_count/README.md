# q003_user_event_count - ユーザー行動ログのイベント件数集計

## 目的
行動ログを読み込み、イベントごとの件数を集計する基礎を身につけることを目的としています。

## 使用技術
Python, csv, 条件分岐, 辞書, 関数

## 実行方法
1. `learning/main.py` を自力で完成させます。
2. `completed/main.py` と見比べて差分を確認します。
3. 必要なら `python learning/main.py` または `python completed/main.py` を実行します。
4. ログデータから件数を数える考え方を README と docs で確認します。

## 学び
- ログ集計はプロダクト分析の基礎であり、まずは件数を正しく数えることが重要です。
