# q009_file_error_handling - ファイル読み込みと例外処理

## 目的
存在しないファイルや不正な数値に対処し、エラーを安全に扱う基礎を身につけることを目的としています。

## 使用技術
Python, Path, csv, try/except, ValueError, FileNotFoundError

## 実行方法
1. `learning/main.py` を自力で完成させます。
2. `completed/main.py` と見比べて差分を確認します。
3. 必要なら `python learning/main.py` または `python completed/main.py` を実行し、正常系と異常系の出力を確認します。
4. どの例外をどこで処理するかを docs と合わせて整理します。

## 学び
- 例外処理を入れることで、想定外の入力があっても原因を把握しやすくなります。
