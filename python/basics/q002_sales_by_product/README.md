# q002_sales_by_product - 商品別売上集計

## 目的
商品ごとの売上合計を辞書でまとめ、売上の大きい順に表示する基礎を身につけることを目的としています。

## 使用技術
Python, csv, 辞書, for文, 関数, sorted

## 実行方法
1. `learning/main.py` を自力で完成させます。
2. `completed/main.py` と見比べて差分を確認します。
3. 必要なら `python learning/main.py` または `python completed/main.py` を実行します。
4. `docs/` を見返し、辞書での集計処理を説明できるようにします。

## 学び
- 辞書で集計する考え方を身につけると、カテゴリ別の集計処理に応用しやすくなります。
