# q002_sales_by_product - 商品別売上集計

    ## 目的
    商品ごとの売上合計を辞書でまとめ、売上の大きい順に表示する。

    ## 使う内容
    辞書, for文, 関数, 並び替え, ファイル読み込み

    ## 学習の進め方
    1. learning/main.py を自力で完成させる
    2. completed/main.py と見比べる
    3. README に「工夫した点」を追記する

    ## 実行例
    Apple: 600円
Orange: 400円
Banana: 400円
Grape: 300円
