# q006_survey_by_age - 年代別の満足度平均

## 目的
年代ごとに平均を出し、グループ別集計の基礎を身につけることを目的としています。

## 使用技術
Python, csv, 辞書, 関数, for文, 条件分岐

## 実行方法
1. `learning/main.py` を自力で完成させます。
2. `completed/main.py` と見比べて差分を確認します。
3. 必要なら `python learning/main.py` または `python completed/main.py` を実行します。
4. 年代ごとの集計方法を README と docs で確認します。

## 学び
- グループ別集計は、分析で対象を比較するための基本パターンです。
