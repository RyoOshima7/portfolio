# q004_conversion_rate - クリック率・購入率の計算

## 目的
view を分母にして click率 と purchase率 を計算し、割合の扱いに慣れることを目的としています。

## 使用技術
Python, csv, 関数, 条件分岐, 例外処理

## 実行方法
1. `learning/main.py` を自力で完成させます。
2. `completed/main.py` と見比べて差分を確認します。
3. 必要なら `python learning/main.py` または `python completed/main.py` を実行します。
4. 割合計算で分母が0になる場合の扱いも確認します。

## 学び
- 割合指標は分析でよく使うため、分母・分子の意味を意識して計算することが大切です。
