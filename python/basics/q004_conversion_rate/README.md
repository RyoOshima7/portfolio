# q004_conversion_rate - クリック率・購入率の計算

    ## 目的
    view を分母にして click率 と purchase率 を計算する。

    ## 使う内容
    関数, 例外処理, 条件分岐, 辞書

    ## 学習の進め方
    1. learning/main.py を自力で完成させる
    2. completed/main.py と見比べる
    3. README に「工夫した点」を追記する

    ## 実行例
    view数: 5
click数: 3
purchase数: 2
click率: 0.60
purchase率: 0.40
