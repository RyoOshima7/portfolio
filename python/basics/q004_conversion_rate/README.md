# q004_conversion_rate - クリック率・購入率の計算

## 目的
view を分母にして click率 と purchase率 を計算し、割合の扱いに慣れることを目的としています。

## 使用技術
Python, csv, 関数, 条件分岐, 例外処理

## 実行方法
1. `learning/main.py` を自力で完成させます。
2. `completed/main.py` と見比べて差分を確認します。
3. 必要なら `python learning/main.py` または `python completed/main.py` を実行します。
4. 割合計算で分母が0になる場合の扱いも確認します。

## 学び
- 割合指標は分析でよく使うため、分母・分子の意味を意識して計算することが大切です。


## BtoB SaaSでどう活かせるか

割合指標を計算する考え方は、BtoB SaaSのファネル分析やKPI確認で重要です。たとえば、無料登録から初回利用、初回利用から継続利用、トライアルから商談化、商談から契約化などの転換率を確認するときに使えます。

この課題の `click率` や `purchase率` は、実務ではCVR、オンボーディング完了率、機能利用率、商談化率などに置き換えて考えられます。分母と分子を正しく定義することが、分析結果の解釈に直結します。

## 面接で説明するなら

割合を出すだけでなく、「分母を何にするか」「どの行動を成功とみなすか」を明確にすることで、施策の効果測定や改善優先度の判断に使えると説明できます。
