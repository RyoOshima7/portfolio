# q010_flask_summary_app - Flaskで集計結果を表示

## 目的
フォーム入力を受け取り、簡単な売上結果を画面表示することで、Webアプリの基本構造を学ぶことを目的としています。

## 使用技術
Python, Flask, ルーティング, フォーム送信, テンプレート表示

## 実行方法
1. `learning/app.py` を自力で完成させます。
2. `completed/app.py` と見比べて差分を確認します。
3. 必要なら対象フォルダで `python completed/app.py` を実行し、ブラウザで表示を確認します。
4. フォーム入力から計算結果表示までの流れを README とコードで確認します。

## 学び
- Flaskの小さなアプリを作ると、入力 → 処理 → 表示の基本的な流れを理解しやすくなります。
