# q008_inventory_order - 在庫・注文管理CLI

## 目的
商品名と注文数を受け取り、在庫を減らす処理を通じて条件分岐と状態更新を学ぶことを目的としています。

## 使用技術
Python, input, if文, while文, 関数, 例外処理

## 実行方法
1. `learning/main.py` を自力で完成させます。
2. `completed/main.py` と見比べて差分を確認します。
3. ターミナルで `python learning/main.py` または `python completed/main.py` を実行し、商品名と注文数を入力して動作を確認します。
4. 在庫不足や商品なしの場合の処理も確認します。

## 学び
- CLI課題では、入力値に応じて処理を分ける考え方を実践できます。
