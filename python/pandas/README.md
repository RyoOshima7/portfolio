# Pandas

## 目的
pandasを使って、CSVなどの表データを読み込み、データ確認・抽出・集計・前処理の基礎を身につけることを目的としています。

## 使用技術
Python, pandas, pathlib, CSV

## 実行方法
1. 各課題フォルダの `docs/problem.md` と `docs/requirements.md` を確認します。
2. `learning/main.py` を実行して、学習途中のコードを確認します。
3. 必要に応じて `completed/main.py` と比較して理解を深めます。
4. `notes/learning_log.md` に、できたこと・分からなかったこと・次にやることを残します。

## 学び
- データ分析では、いきなり集計する前に、まずデータの中身・列名・データ型・欠損値を確認することが重要です。
- 小さな課題として記録することで、pandasで何ができるようになったかをGitHub上で説明しやすくなります。
