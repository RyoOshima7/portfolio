# ユーザー行動ログの集計

## 目的

ユーザー行動ログから行動種類別の発生件数を集計し、ユーザー行動の傾向を把握することを目的とした。

## 処理の流れ

pandasでCSVを読み込み、head()で内容を確認した後、value_counts()でevent列を行動種類別に集計した。

 ## 結果から何が言えるか

 100件の行動ログのうち、viewが43件で最も多く、次にclickが23件、purchaseが19件だった。
 今回のデータでは、閲覧行動が最も多く記録されている。

 ## 学んだこと

行動別の件数を比べることで、ユーザー行動の全体傾向を把握し、次に詳しく分析する箇所を絞り込めることを学んだ。

 ## BtoB SaaSでどう生かせるか

 ユーザー行動別の件数を集計する方法は、BtoB SaaSにおける機能利用や登録・購入・解約などの行動傾向を把握し、詳しく分析する箇所の候補を見つける基礎分析に生かせる。

 ## 次に確認したいデータは何か

 企業・組織を識別するアカウントIDや契約プラン、流入経路を追加し、どのような顧客が購入やキャンセルに至っているか確認したい。
