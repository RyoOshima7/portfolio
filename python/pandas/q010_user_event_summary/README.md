# ユーザー行動ログの集計

 ## 何を分析したか

 user_events.csvを使用し、value_counts()で行動種類別の発生件数を集計した。

 ## 結果から何が言えるか

 100件の行動ログのうち、viewが43件で最も多く、次にclickが23件、purchaseが19件だった。
 今回のデータでは、閲覧行動が最も多く記録されている。

 ## BtoB SaaSでどう生かせるか

 ユーザー行動別の件数を集計する方法は、BtoB SaaSにおける機能利用や登録・購入・解約などの行動傾向を把握し、詳しく分析する箇所の候補を見つける基礎分析に生かせる。

 ## 次に確認したいデータは何か

 企業・組織を識別するアカウントIDや契約プラン、流入経路を追加し、どのような顧客が購入やキャンセルに至っているか確認したい。
