# 月別売上の集計

## 目的

注文データから注文ごとの売上金額を計算し、月別の合計売上を集計することを目的とした。

## 処理の流れ

pandasでCSVを読み込み、head()とinfo()でデータを確認した。
date列を日付型に変換し、数量と単価を掛けて売上金額を計算した後、groupby()を使って月別に集計した。

## 分かったこと

2026年4月の売上が438,400円で最も多く、次に5月の416,980円が多かった。
一方、6月は88,120円で最も少なく、月によって売上に大きな差があることが分かった。

## 次に調べたいこと

4月と5月の売上が多くなった原因を確認するため、商品カテゴリ別または商品別に売上を集計したい。

## BtoB SaaSでどう生かせるか

月別に数値を集計する方法は、BtoB SaaSにおけるMRR、契約件数、機能利用回数などの推移を確認する分析に応用できる。
