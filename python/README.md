# Python

## 目的

## 使用技術
Python, pandas, Flask, CSV処理, 例外処理, Markdown

## 実行方法
1. `basics/` でPython基礎課題を確認します。
2. `pandas/` でCSV読み込み・データ確認・集計などのデータ処理課題を確認します。
3. 必要に応じて `data_analysis/` で集計・前処理・可視化の課題を追加します。
4. 各課題は README を見れば目的と進め方が分かる構成にします.
