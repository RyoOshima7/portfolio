# q003_if_and_summary - IF 判定列 + summary 整形

## 目的
スプレッドシートで、表の確認・基本集計・条件付き集計の基礎を身につける。

## 使った内容
IFで条件判定列を作る, COUNTIFで1条件の件数を集計する, COUNTIFSで複数条件の件数を集計する

## 学習の進め方
1. `data/raw_data.csv` を開いて元データを確認する
2. `learning/summary_learning.xlsx` を自力で作成する
3. `docs/problem.md` と `docs/requirements.md` を読み、何を確認した課題か説明できるようにする
4. `notes.md` に、学んだことと実務へのつながりを残す

## 学んだこと
- IFで元データに判定列を追加すると集計しやすくなることを学んだ
- COUNTIF / COUNTIFSで条件付き集計の基礎を確認できた
- 一度修正を入れながら、参照範囲を固定して数える重要性を理解できた

## 実務とのつながり
条件ごとの件数集計や、簡単なKPI集計・ステータス分類の基礎として使える。
