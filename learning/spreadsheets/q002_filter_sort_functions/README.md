# q002_filter_sort_functions - フィルタ / 並び替え / 基本関数

## 目的
スプレッドシートで、表の確認・基本集計・条件付き集計の基礎を身につける。

## 使った内容
フィルタで条件に合う行を確認する, 並び替えで値の大小を見やすくする, SUM / AVERAGE / COUNT / COUNTAで集計する

## 学習の進め方
1. `data/raw_data.csv` を開いて元データを確認する
2. `learning/summary_learning.xlsx` を自力で作成する
3. `docs/problem.md` と `docs/requirements.md` を読み、何を確認した課題か説明できるようにする
4. `notes.md` に、学んだことと実務へのつながりを残す

## 学んだこと
- フィルタと並び替えでデータの見え方が大きく変わることを学んだ
- 基本関数で売上や件数を短時間で要約する練習ができた
- Summaryに必要な数値をまとめる流れを理解できた

## 実務とのつながり
売上確認、件数把握、簡易レポート作成など、日常的な表計算業務の土台になる。
