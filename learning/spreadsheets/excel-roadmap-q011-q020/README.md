# excel-roadmap-q011-q020 - Excel基礎（VLOOKUP / 絶対参照 / SUMIFS / COUNTIFS / AVERAGEIF / ピボットテーブル）

## 目的
Excel未経験〜初心者向けに、分析職の土台となる **参照関数・複数条件集計・絶対参照・ピボットテーブル** の基礎を身につけることを目的としています。

## 使用技術
Excel, VLOOKUP, 絶対参照（$）, SUMIFS, COUNTIFS, AVERAGEIF, ピボットテーブル

## 実行方法
1. `workbook/excel-roadmap-q011-q020.xlsx` を開きます。
2. `orders_data` シートで青文字セルに数式を入力します。
3. q020 は `orders_data` を元にピボットテーブルを作成します。
4. `pivot_task` シートに指定された値を転記します。
5. `practice` シートで q011〜q020 がすべて `OK` になるまで修正します。
6. `docs/problem.md` と `docs/solutions.md` を見返し、参照や集計の考え方を整理します。
7. `notes/learning_log.md` に学習内容・つまずき・実務とのつながりを記録します。

## 学び
- 参照関数と絶対参照を理解すると、複数の表をまたいだ集計がしやすくなります。
