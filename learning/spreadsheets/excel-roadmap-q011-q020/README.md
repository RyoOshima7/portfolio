# excel-roadmap-q011-q020 - Excel基礎（VLOOKUP / 絶対参照 / SUMIFS / COUNTIFS / AVERAGEIF / ピボットテーブル）

## 目的
Excel未経験〜初心者向けに、分析職の土台となる **参照関数・複数条件集計・絶対参照・ピボットテーブル** の基礎を身につける。

## 使った内容
`VLOOKUP`, 絶対参照（`$`）, `SUMIFS`, `COUNTIFS`, `AVERAGEIF`, 割引後売上計算, 税込売上計算, ピボットテーブル

## 学習の進め方
1. `workbook/excel-roadmap-q011-q020.xlsx` を開く
2. `orders_data` シートで青文字セルに数式を入力する
3. q020 は `orders_data` を元にピボットテーブルを作成する
4. `pivot_task` シートに指定された値を転記する
5. `practice` シートで q011〜q020 がすべて OK になるまで修正する
6. `docs/problem.md` と `docs/solutions.md` を見返しながら、何を学んだか説明できるようにする
7. `notes/learning_log.md` に学習内容・つまずき・実務とのつながりを残す

## 学んだこと
- `VLOOKUP` でマスタ表からカテゴリや単価を参照できる
- 絶対参照を使うと、コピーしても固定したいセルをずらさずに使える
- `SUMIFS` / `COUNTIFS` / `AVERAGEIF` で条件に応じた集計ができる
- ピボットテーブルを使うと、地域×カテゴリの売上を素早く要約できる

## 実務とのつながり
商品マスタ参照、地域別の条件付き集計、税率の固定参照、売上要約表の作成など、分析業務やレポート作成でよく使う表計算スキルにつながる。
