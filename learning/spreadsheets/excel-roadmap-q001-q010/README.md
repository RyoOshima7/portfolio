# excel-roadmap-q001-q010 - Excel基礎（売上計算 / 基本集計 / 条件付き集計 / IF）

## 目的
Excel未経験〜初心者向けに、分析職の土台となる **売上計算・合計・平均・件数・条件付き集計・判定列作成** の基礎を身につける。

## 使った内容
数量×単価の計算, `SUM`, `AVERAGE`, `COUNTA`, `COUNTIF`, `COUNTIFS`, `SUMIF`, `IF`

## 学習の進め方
1. `workbook/excel-roadmap-q001-q010.xlsx` を開く
2. `sales_data` シートで青文字セルに数式を入力する
3. `practice` シートで q001〜q010 がすべて OK になるまで修正する
4. `docs/problem.md` と `docs/solutions.md` を見返しながら、何を学んだか説明できるようにする
5. `notes/learning_log.md` に学習内容・つまずき・実務とのつながりを残す

## 学んだこと
- Excelではセル参照を使って計算できる
- `SUM` と `AVERAGE` で基本集計ができる
- `COUNTIF` / `COUNTIFS` / `SUMIF` で条件付き集計の基礎を実践できる
- `IF` で判定列を作ると、集計しやすい形にデータを整えられる

## 実務とのつながり
売上表の初歩的な確認、担当者別・カテゴリ別の集計、条件に応じた判定列作成など、分析業務の入口になる表計算スキルにつながる。
