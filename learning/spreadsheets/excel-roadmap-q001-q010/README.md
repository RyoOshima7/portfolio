# excel-roadmap-q001-q010 - Excel基礎（売上計算 / 基本集計 / 条件付き集計 / IF）

## 目的
Excel未経験〜初心者向けに、分析職の土台となる **売上計算・合計・平均・件数・条件付き集計・判定列作成** の基礎を身につけることを目的としています。

## 使用技術
Excel, セル参照, SUM, AVERAGE, COUNTA, COUNTIF, COUNTIFS, SUMIF, IF

## 実行方法
1. `workbook/excel-roadmap-q001-q010.xlsx` を開きます。
2. `sales_data` シートで青文字セルに数式を入力します。
3. `practice` シートで q001〜q010 がすべて `OK` になるまで修正します。
4. `docs/problem.md` と `docs/solutions.md` を見返し、各関数の意味を説明できるようにします。
5. `notes/learning_log.md` に、学習内容・つまずき・実務とのつながりを記録します。

## 学び
- セル参照を使うと、手計算ではなく表全体を効率よく扱えます。
