# q001_csv_check - 列名 / 型 / 空白の確認

## 目的
スプレッドシートで、表の確認・基本集計・条件付き集計の基礎を身につける。

## 使った内容
列名を確認する, 数値列と文字列列を分けて見る, COUNTBLANKで空白セル数を確認する

## 学習の進め方
1. `data/raw_data.csv` を開いて元データを確認する
2. `learning/summary_learning.xlsx` を自力で作成する
3. `docs/problem.md` と `docs/requirements.md` を読み、何を確認した課題か説明できるようにする
4. `notes.md` に、学んだことと実務へのつながりを残す

## 学んだこと
- CSVを開いたときは、まず列名・型・空白を確認する重要性を学んだ
- Summaryシートで確認観点を表に整理する練習ができた
- 集計前にデータの状態を把握する意識が身についた

## 実務とのつながり
データを受け取った直後の初期確認や、集計前の前処理チェックにつながる。
