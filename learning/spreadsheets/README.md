# Spreadsheets

## 目的
スプレッドシートの基礎を、小課題を通じて身につけ、分析職で必要な集計・参照・可視化の土台を作ることを目的としています。

## 使用技術
Excel, 関数（SUM / IF / COUNTIF / COUNTIFS / SUMIFS / AVERAGEIF / VLOOKUP）, ピボットテーブル

## 実行方法
1. `excel-roadmap-q001-q010/` で基本計算、条件付き集計、IFの基礎を学びます。
2. `excel-roadmap-q011-q020/` で参照関数、絶対参照、複数条件集計、ピボットテーブルを学びます。
3. 各フォルダの `workbook/` を解き、`docs/` と `notes/` を見ながら復習します。

## 学び
- 表計算は、関数を覚えるだけでなく、どの表をどう参照して集計するかを理解することが大切です。
