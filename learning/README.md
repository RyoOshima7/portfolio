# Learning（基礎）

## 目的
数学、統計、基本情報、表計算などの基礎学習を、後から見返せるProofとして残すことを目的としています。

## 使用技術
Markdown, 数学基礎, 統計基礎, FE試験学習, スプレッドシート

## 実行方法
1. `math/` `statistics/` `fe_exam/` `spreadsheets/` の順に確認します。
2. 各フォルダでノート、例題、課題を見て、学習の流れを把握します。
3. 新しい内容を追加する際は、概念だけでなく小さな例題も残します。

## 学び
- 基礎学習は読んで終わりではなく、例題や再現を通じて定着させることが大切です。
